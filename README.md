# RC 系统前端
* 项目地址：https://risk-demo.corp.dianrong.com

# Getting Started
```
// 连接YAPI 
npm install
npm start
```
或者
```
// 连接Demo环境
npm install
npm start:dev
```
# 生成代码
```sh
npm run code
```

# Build && Deploy
```
// Demo环境
npm run build-demo
npm run deploy
```

```
// 生产环境
// Demo环境
npm run build
npm run deploy
```
